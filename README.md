# E-Commerce Store
In this online store, you're able to see Products and Collections or create a
shopping cart without signing up. After
You log in, You can create an order.
Users can filter products based on
price and collection, Search them by
name and description and sort them by
price.

## Installation
- Clone the project:
```
git clone https://github.com/MohamadrezaPiri/E-Commerce-DRF.git
```
- Create a virtual environment:
```
py -m venv YourVirtualEnvironment
```
- Then activate it:
```
YourVirtualEnvironment/Scripts/activate
```
- Using the command below, install all the packages in ```requirements.txt``` file:
```
pip install -r requirements.txt
```
- And finally:
```
py manage.py migrate
```
- Now it's time to run the server:
```
py manage.py runserver
```
